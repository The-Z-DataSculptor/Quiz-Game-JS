🚀 Interactive Quiz Game
Built by Zain Hassan
A sleek, responsive, and modern Quiz Game built with Vanilla JavaScript, HTML5, and CSS3. This project features a dynamic scoring system, a custom progress bar, and a "mobile-first" design.

🌟 Key Features
Dynamic UI: Questions and answers are injected into the DOM via JavaScript—no page reloads required.

Real-time Feedback: Buttons change color (Green/Red) immediately to show correct or incorrect choices.

Custom Progress Tracking: A visual progress bar that updates as the user moves through the quiz.

Responsive Design: Fully optimized for desktops, tablets, and smartphones using CSS Flexbox and Grid.

Smooth Animations: Custom @keyframes for screen transitions and score celebrations.

🛠️ Technologies Used
HTML5: Semantic structure for the game screens.

CSS3: Custom variables, linear gradients, and cubic-bezier transitions for a premium feel.

JavaScript (ES6): Array manipulation, event listeners, and DOM management.
